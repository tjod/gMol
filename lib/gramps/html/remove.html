REMOVE

Objects can be removed from groups by using the remove command. The effect of this command is to remove the object from all grouping and place it at the end of the display list. The syntax of the remove command is:

remove ANYNAME

where ANYNAME is any object or group name. Of course, the display on the screen reflects the changes made to the tree by the remove command.

/menu

/menu - Causes specified objects to be removed from the menu. The syntax of this command is:

remove/menu name [,name,name...]

where name is any GRAMPS object name. Multiple names may be given on one line. The effect of this command is to remove the named object from the menu and reinsert it into the World. If the item had been a member of a group at the time it had been inserted into the MENU, it will not be returned to that group. It will be put at the end of the TREE.
