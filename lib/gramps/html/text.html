TEXT

The text command is used to draw text characters on the screen. The syntax of the command is:

text STRING name value,value,value

STRING is any string of one or more characters. If STRING is to contain DELIMITERs (blanks or commas) it must be enclosed in double quotation marks. If the string contains lower case letters, even if it contains no DELIMITER characters, it should also be enclosed in double quotation marks to ensure no translation to upper case is done. name must be a new name not already known to the system.

The values are interpreted as X, Y, and Z coordinates for placement of the lower left corner of the first character in STRING. These values are optional. For the values which are omitted, an assumption is made. If the X or Y coordinate is missing, then the default value of 0 is used. If the Z coordinate is missing, the default value used is 0.
