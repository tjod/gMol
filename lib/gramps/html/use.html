USE

The use command uses two of the previously picked points. The syntax of the command is:

use #N1, #N2 name

where N1, N2 and N3 are point variables (see pick/point) The # symbol is a required part of the syntax of this command. The effect of the command is to refer the arbitrary axis R to the axis specified by the endpoints N1 and N2. Using axis R in any rotate command will cause the picture to rotate around the arbitrary axis that the picture is currently USing.
